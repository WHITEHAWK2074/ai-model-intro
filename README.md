# AI大模型科普网页
面向AI入门者的静态科普网页，前端入门练手项目，基于 **HTML5+CSS3+Bootstrap** 开发，全响应式布局，科普三款主流大模型核心知识。

## 在线预览
https://whitehawk2074.github.io/ai-model-intro/

## 项目介绍
前端基础练手作品，将AI大模型专业知识通俗化呈现，覆盖大模型基础认知 + DeepSeek、OpenClaw、Claude三款模型专项解析，适配电脑、平板、手机全终端。

## 核心特性
- 全响应式布局，基于Bootstrap实现多终端自适应
- 知识体系清晰，从通用概念到专项解析层层递进
- 精准区分三款模型核心定位与应用场景
- 轻量简洁，代码结构清晰易学习、易修改

## 技术栈
- 基础：HTML5 + CSS3
- 框架：Bootstrap
- 布局：响应式网格、弹性布局
- 部署：GitHub Pages

## 本地运行
```bash
# 克隆仓库
git clone https://github.com/WHITEHAWK2074/ai-model-intro.git

# 进入目录
cd ai-model-intro

# 浏览器打开 index.html 即可运行
```

## 页面内容
1. **AI大模型综合认知**
   核心定义、训练逻辑、发展趋势、通用应用价值

2. **主流模型解析**
   - DeepSeek：国产轻量化文本大模型，低算力高性能
   - OpenClaw：开源具身智能框架，支持指令转实操动作
   - Claude：长上下文、强推理、高安全性专业大模型

## 项目说明
纯前端静态页面，无复杂交互，适合学习Bootstrap响应式开发、页面结构设计。
